# Отчёт ЛР12 — Интерфейсы и хранилища

**Автор:** Иванов Иван  
**Класс:** 10-А  

## Цель
Добавить интерфейс хранения задач и возможность подмены хране
ния (JSON/Memory).

## Что сделано
- Создан интерфейс `ITaskStorage` с методами Load/Save  
- Реализован `MemoryTaskStorage` (данные в памяти)  
- Реализован `JsonTaskStorageAdapter` (адаптер для JSON)  
- В App при запуске можно выбрать режим хранения  

## Проверка
1. Выбрать JSON режим, добавить задачу → перезапуск → задача осталась  
2. Выбрать Memory режим, добавить задачу → перезапуск → список пуст
